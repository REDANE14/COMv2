WINDOWS 10 INSTALLATION

Instruction

Tips: Make sure your computer meets the minimum hardware specifications listed by Microsoft for the Windows version you're installing before beginning the Windows OS installation.

  1. Insert the USB drive: Insert the bootable USB drive into your computer's USB port.
  2. Boot from USB drive: Restart your computer and enter the BIOS setup. Change the boot order so that the computer boots from the USB drive first.
  3. Begin the Windows installation process: Once your computer has successfully booted from the USB drive, you should see the Windows 10 setup screen. Click the "Install Now" button to begin the installation process.
  4. Enter the product key: Enter the product key for Windows 10 when prompted. You can either enter the product key now or skip this step and enter it later.
  5. Choose the installation type: Select the "Custom" installation type to perform a clean installation of Windows 10.
  6. Select the hard drive to install Windows 10: Choose the hard drive where you want to install Windows 10. You can create new partitions or delete existing ones if needed.
  7. Begin the installation process: Click the "Next" button to start the installation process. Windows 10 will now be installed on your computer.
  8. Set up Windows 10: Once the installation process is complete, you will be prompted to set up Windows 10. Follow the on-screen instructions to set up your user account and personalize your settings.
