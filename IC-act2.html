1.2 PC Assembly

Instruction

Tips: Make sure to ground yourself and handle components with care to avoid damaging them, and use a reputable thermal paste to ensure proper heat transfer between the CPU and its cooler for optimal performance and stability.

  1. Open the case: Unscrew the side panel.
  2. Install the PSU: Position the fan facing down.
  3. Install the CPU: Lift the lever and match the golden triangle.
  4. Apply thermal paste: Put a grain size thermal paste.
  5. Attach the CPU cooler.
  6. Place the motherboard: Put the I/O shield before the motherboard.
  7. Plug the case connectors: USB 2.0-3.0, and Front Panel.
  8. Install Storage device: SATA cable.
  9. Plug PSU cables: ATX power connection, and hard drives.
